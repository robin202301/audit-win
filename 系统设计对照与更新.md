## 二、当前实现架构总览

### 2.1 技术栈

- **主进程**: Electron + TypeScript (tsc && tsc-alias 编译)
- **渲染进程**: Vue 3 (Composition API) + Vite + Pinia + Vue Router
- **数据库**: SQLite (better-sqlite3)
- **文档处理**: docxtemplater + pizzip（模板填充）、mammoth.js（Word 解析）
- **打包**: electron-builder → release/ 目录

### 2.2 审计类型（仅三种）

1. **经济责任审计**
2. **预算执行审计**
3. **专项审计调查**

### 6.2 背景纹理与水印

全局背景使用 CSS `repeating-linear-gradient` 模拟华表柱网格纹理：
```css
background: #f5f0e8;
background-image:
  repeating-linear-gradient(90deg, transparent, rgba(180,40,20,0.03) 48px),
  repeating-linear-gradient(0deg, transparent, rgba(180,40,20,0.02) 48px);
```

所有页面通过 `body::before` 固定定位伪元素叠加华表柱水印背景：
- 两侧径向渐变模拟华表柱轮廓
- 垂直条纹重复纹理模拟柱体
- 祥云装饰点增加传统元素
- `z-index: 0` 确保在内容下方显示
